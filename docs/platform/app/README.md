# App Platform Docs

Use these docs when adding or refactoring an app module.

| Need | Read |
|---|---|
| Required files, manifest fields, entry component, hooks, resources, UI rules | [`module-contract.md`](module-contract.md) |
| Resource files: icons, colors, strings, dimensions | [`resources.md`](resources.md) |
| Status/nav bars, keyboard resize, pointer events, URL-driven UI state | [`ui-shell.md`](ui-shell.md) |
| Constants, defaults, world data, runtime overlay | [`data-layering.md`](data-layering.md) |
| Navigation declarations and graph generation | [`../navigation/declaration.md`](../navigation/declaration.md) |
| App runtime state, world data, snapshots | [`../state/model.md`](../state/model.md) |
| Cross-app calls from or into the app | [`../os/intent-system.md`](../os/intent-system.md) and [`../os/cross-app-launch.md`](../os/cross-app-launch.md) |

This directory is app-author-facing. OS internals live under [`../os/`](../os/).
