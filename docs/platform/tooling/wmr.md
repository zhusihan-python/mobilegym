# WMR Tooling

WMR is part of the platform tooling surface for rendering XML-like UI/resource structures. The detailed implementation notes live in [build.md](build.md) and `os/wmr/`.

## When It Matters

Use this page as the lookup when changing:

| Area | Path |
|---|---|
| WMR renderer | `os/wmr/WmrRenderer.tsx` |
| WMR parser | `os/wmr/engine/parser.ts` |
| Generated/rendered resources | App/system resource pipelines that feed WMR |
